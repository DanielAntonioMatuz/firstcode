Reglas para variables

Crear una variable en UPWIND, tiene ciertas reglas a seguir, un poco distinto a los diferentes lenguajes de programación, para ello, primero(1), deberemos elegir un tipo de dato, una vez elegido, el nombre de la variable podrá tener el nombre que desee, únicamente no combine números (no permitido), simbolos a excepción del guión bajo (_) y espacio entre ellos.
